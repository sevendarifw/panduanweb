Troubleshooting

Menggunakan Debug Panel

Debug Panel

Activities: ‚úÖ Loaded (15)
BPPs: ‚ùå Failed

‚ùå Masalah Umum

Data tidak muncul:
  • URL Google Sheets salah
  • Sheet belum dipublish
  • Format URL bukan CSV
Data "undefined":
  • Nama kolom tidak sesuai
  • Ada spasi ekstra di header
  • Format data tidak konsisten
BPP tidak bisa diklik:
  • Data BPP belum termuat
  • JavaScript error
  • Nama BPP tidak match

‚úÖ Solusi

Cek URL:
  • Pastikan berakhiran &output=csv
  • Test buka URL di browser baru
  • Harus menampilkan data CSV
Cek Format Data:
  • Header sesuai panduan
  • Tidak ada spasi ekstra
  • Nama BPP konsisten
Gunakan Debug Panel:
  • Klik "Show Available Columns"
  • Bandingkan dengan yang diharapkan
  • Refresh data jika perlu
